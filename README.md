# Casilda.ASPACE.2026.Julio
Guía interactiva móvil para cuidadores de Casilda en Aspace Sevilla. Incluye medicación, comidas, baño, descanso, corsé y férulas, ocio, precauciones y preguntas frecuentes. Botones de llamada directa a los padres siempre visibles. Diseñada para consulta rápida durante la atención diaria.
